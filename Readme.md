# 🎓 Premier Schools Exhibition – Landing Page

A fully responsive and accessible static landing page built using **HTML5, CSS3, and JavaScript**, based on the provided Figma design.

---

## 🚀 Overview

This project is a pixel-perfect implementation of the **Premier Schools Exhibition (PSE)** landing page.
It focuses on **clean structure, accessibility, responsiveness, and smooth UI interactions** without using any external frameworks.

---

## 🛠️ Tech Stack

- HTML5 (Semantic Markup)
- CSS3 (Custom styling with BEM methodology)
- Vanilla JavaScript (for interactions & sliders)
